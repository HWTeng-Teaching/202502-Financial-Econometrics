# (a) 設定虛無假設(_H<sub>0</sub>_)和對立假設(_H<sub>1</sub>_)
- 虛無假設(_H<sub>0</sub>_)：GDP和奧運獎牌數無關，即 H<sub>0</sub>：β₂ = 0  
- 對立假設(_H<sub>1</sub>_)：GDP和奧運獎牌數有正向關係，即H<sub>1</sub>：β₂ > 0  
- 說明：右尾檢定（right-tailed test），因為假設 GDP 增加會導致獎牌數增加
